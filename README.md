## Requirements
- Python 3.x
- SQLite Database

## Install Dependencies

1. Create virtual environment
```
$ python -m venv js-blog
$ source js-blog/bin/activate
```

2. Install via pip
```
$ pip install -r requirements.txt 
```

## How to Run

In project folder, run
```
$ flask run
```
Or
```
$ python app.py
```
